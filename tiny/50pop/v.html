Installing OpenBSD 6.0 on Vultr

  1. Sign up or log in at vultr.com. Click ISO from the top menu, then [Add ISO]
  2. Paste in http://ftp3.usa.openbsd.org/pub/OpenBSD/6.0/amd64/install60.iso (or closer mirror) and click [Upload]
  3. Wait a minute for it to finish uploading, until Status says Available.
  4. Click Instances in the top menu, then deploy one. Any server location.
  5. Under Server Type, choose Custom ISO, and click your [install60.iso] box so it's blue.
  6. Any server size OK. No additional features. Any hostname and label. Then [Deploy Now] button at bottom. It takes a couple minutes. Go take a break.
  7. When Status says Running, click the name of your server to get into Server Details
  8. In the top-right corner of the screen, click the icon that looks like a monitor to View Console.

That should pop up a new window with a black screen OpenBSD installation program. But if not, go back and do these two additional steps:

  1. Click Settings on the top menu, then Custom ISO on the left. It probably says “Current ISO: Mounting error: Unexpected exit code: 255”.
  2. Click [Attach ISO and Reboot]

Then go to the installation guide.
